# Valen Corretora de Seguros - Landing Page

Landing page mobile-first para venda de planos de saúde individual, familiar e empresarial.

## Publicar no GitHub Pages

1. Envie estes arquivos para um repositório.
2. Vá em Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main / root.

## Alterar WhatsApp

No arquivo `script.js`, altere:

```js
const WHATSAPP_NUMBER = "5511971058091";
```

## Eventos para GTM

- `whatsapp_click`
- `lead_form_whatsapp`
